## Per-channel publish order

Suggested sequence on drop day (Eastern time as default — adjust for your audience):

1. **T-2h** — Pin the IG grid carousel (5 slides) early to start building anticipation.
2. **T-30m** — Publish IG Stories 1–3 with countdown running + poll bait live.
3. **T-0** — Publish Story 4 + 5 with link sticker live, drop the X thread, pin the C-variant IG caption as a teaser comment on the grid post.
4. **T+1h** — Tweet C-variant + Story reply with live stock count.
5. **T+6h** — Post a "ticker" Story: "1 of 2 remaining" if not yet sold.
6. **T+12h** — If still live, push Variant C with the 2/2 teaser hook to seed the next release.
