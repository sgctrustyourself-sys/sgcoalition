TEMPLATING CHECKLIST · COPY ADDITIONS

6 copy-edit checks
  • Replace every {{release_name}}, {{x}}/{{y}}, {{price}}, {{shop_url}} placeholder.
  • Drop in the right hashtags from the bank (#{{release_hashtag}} is the only one that changes per drop).
  • Confirm character counts: IG caption ≤ 2,200; X single ≤ 280; each X thread tweet ≤ 280; Slack one-liner ≤ 200.
  • Cross-check copy against drop-kit Slide 3 ({{x}} OF {{y}}, {{price}}) and Slide 5 ({{shop_url}}) — copy must match the slide lockup verbatim.
  • Re-read caption aloud once before pasting. Streetwear copy that doesn't read as a sentence fails engagement.
  • Story copy stack has safe-zone clearance — no text crosses into the IG profile handle (top 250 px) or CTA tray (bottom 320 px).

PER-CHANNEL PUBLISH ORDER

T-2h → T+12h Eastern time default · adjust for audience

Suggested sequence on drop day (Eastern time as default — adjust for your audience):

  1. T-2h Pin the IG grid carousel (5 slides) early to start building anticipation.
  2. T-30m Publish IG Stories 1–3 with countdown running + poll bait live.
  3. T-0 Publish Story 4 + 5 with link sticker live, drop the X thread, pin the C-variant IG caption as a teaser comment on the grid post.
  4. T+1h Tweet C-variant + Story reply with live stock count.
  5. T+6h Post a "ticker" Story: "1 of 2 remaining" if not yet sold.
  6. T+12h If still live, push Variant C with the 2/2 teaser hook to seed the next release.
