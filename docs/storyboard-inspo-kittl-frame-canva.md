## Kittl · primary pick · still IG Story art

[Kittl IG Story templates](https://www.kittl.com/templates/instagram-stories)

Kittl is the right call when you want typography + texture control that Canva can't match. Re-skin workflow is one click per layer: drop the background to `#050505`, swap fonts to `Bebas Neue` (display) + `Inter` (body) — both available in the Kittl font library. Save the 5 slides as one project + use `Kittl Flows` to lock the brand kit for future drops.

### Search queries to copy-paste · Kittl

```
Kittl search bar:
  Editorial           → 10+ matches · filter "Story"
  Minimalist          → 30+ matches · filter "Story"
  Lookbook            → 5+ curated fashion packs · 5–10 slides per set
  Product Launch      → 8+ packs with countdown slots built-in

  DO NOT search: streetwear              // over-styled OR neon — clashes with deep-black restraint
              playful                  // rounded corners + drop shadows
              pop / bright            // violates limited-edition tension
```

> **Kittl re-skin recipe · 5 minutes per slide:**
> ① background layer → set to `#050505`.
> ② display text → swap to `Bebas Neue` 88–140px white.
> ③ body text → swap to `Inter` 24–32px `#C8C8C8`.
> ④ add overlay gradient rectangle (top-left black 30% → transparent) for wordmark safe-zone.
> ⑤ export PNG @2x for IG.

### Canva-specific gotcha · IG stickers

> Canva does **NOT** host native IG interactive stickers (countdown / poll / link / mention). In your layout, drop a low-opacity rectangle ghost box labeled `[Sticker landing zone]` where the IG sticker will go — then place the live sticker in IG at publish time. This isn't a bug, it's IG behavior. The ghost box is your visual contract with the sticker tray.

### Search queries to copy-paste · Canva

```
  black minimalist business  → free B&W generic → strip + condense

  DO NOT search: streetwear / urban  → neon OR paint splatter
              playful / bright      → clashes with 1-of-2 tension
```

## Motion · for animated Stories / Reels

If your Story needs kinetic typography, slow-zoom product reveals, or vignette transitions — Kittl and Canva aren't enough. Use these motion platforms. All three let you set the global hex color to `#050505` across all slides so the brand surface stays unified.

| Platform | URL | Format | Slide roles |
|---|---|---|---|
| **Motion Array** | [motionarray.com](https://motionarray.com) · search: `dark-fashion-story` | After Effects + Premiere project files | HERO + MANIFESTO |
| **Envato Elements** | [elements.envato.com](https://elements.envato.com) · search: `dark-luxury-fashion-story-vertical` | Subscription · AE + video + photo graphics | DETAIL + CTA |
| **Jitter.video** | [jitter.video](https://jitter.video) · search: `dark fashion story` | Browser-based · rapid iteration | HERO + SCARCITY |

### 5-slide motion mapping (rough spec)

- **Hero · slow-zoom:** front.jpeg 1.0 → 1.08 scale over 3s · easing ease-out · audio: optional sub-bass swell
- **Detail · film-strip:** back.jpeg + charcoal gradient fade-up · bottom 35% `#000` 0 → 70% opacity sweep · 2s
- **Scarcity · glitch:** top half front + countdown ticks · IG native sticker handles the countdown — let it
- **Manifesto · kinetic type:** "TRUST YOURSELF." fade-in letter-by-letter · each letter 0.05s delay · final letter holds 4s
- **CTA · click-pulse:** front 60% + arrow pulse + link sticker · arrow brand mark `→` opacity pulse 1.0 → 0.7 → 1.0 × 3 loops

> **When to use motion vs still:** motion is essential for Reels + Sponsored Story ads. For organic Story drops, a still slide is usually enough — motion on every slide reads as over-produced. Use motion for the Hero (slide 1) + the CTA (slide 5); keep Detail + Scarcity + Manifesto still.

## Ship tonight · 5-step workflow

1. **Pick one platform.** Kittl if you have a designer-block budget. Canva if you don't. Motion Array only if you're shipping a Reel/Sponsored ad — not for organic Story drops.
2. **Find or build a 5-slide template.** Either pick from **Group A** above, or build one in the platform using the layer specs from [drop-kit](../drop-kit-grey-wave.html) Slides 1–5.
3. **Re-skin globally.** Background → `#050505`. Display fonts → `Bebas Neue` or `Oswald`. Body → `Inter`. Outline / hairline accents → `rgba(255,255,255,0.08)`.
4. **Drop the copy.** Pull from [drop-copy](../drop-copy-grey-wave.html): IG caption long for slide 1, IG Story sequence copy for slides 1–5, Slack one-liner for the bio link.
5. **Export PNG → publish to IG.** Export each slide as PNG @2x. In IG, drop each PNG into a Story, then add live stickers (countdown / poll / mention / link) over the ghost zones for slides 1, 2, 3, 5.

> **Total time:** 40 minutes if the wallet PNGs are already on disk (`public/images/grey-wave-wallet-1-2-{front,back}.png`). 90 minutes if you also need to re-scan sizes via `python docs/tmp/make-per-format-previews.py --scan-sizes`.
