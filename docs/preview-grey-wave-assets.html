Reviewer checklist · pre-publish

  1. All slides — copy stack has safe-zone clearance: nothing crosses into the top 250 px or bottom 320 px.
  2. All slides — live preview on actual Android/iOS IG account before publishing (treat Canva preview as soft only).
  3. All slides — thread publishes in this exact order: Hero → Detail → Scarcity → Manifesto → CTA.
  4. Story 1 · Hero — countdown end-time matches launch time exactly; @sgcoalition mention resolves.
  5. Story 2 · Detail — back.jpeg has #000 black overlay at bottom 35% before any copy is placed.
  6. Story 3 · Scarcity — countdown end-time in IG-recognised timezone; poll "Should we run 2/2? / Yes / Wait" pulling demand signal.
  7. Story 4 · Manifesto — wallet texture at 10% opacity never competes with "TRUST YOURSELF." copy.
  8. Story 5 · CTA — link sticker URL == PDP URL (no trailing slash, no redirect typo).
  9. Grid 1-5 — copy legibility at thumbnail size in-carousel (24 px body needs to read).
  10. Grid 5 — caption + bio link do the CTA conversion (link sticker NOT supported on Grid).
  11. X 1-4 — tweets publish in thread order (greedy-engagement reads sequentially, brand beat lands last).
  12. X 2 · scarcity — primary urgency driver for X feed, must read before "$35" in visual order.
